# MovieApp

Example app showing list of popular movies loaded from themoviedb.org and their details. It is possible to search movies. 

Used technologies:
- Flow
- Koin
- Retrofit
- LiveData
- Picasso library
- MVVM architecture
- Epoxy for RecyclerView
- Endless scrolling
